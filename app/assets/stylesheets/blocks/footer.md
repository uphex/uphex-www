# Footer

A footer is a block element that should be placed at the bottom of the page.

```
<footer class="footer">
  <div class="container">
    <a class="footer__title" href="/">UpHex</a>
    <ul class="footer__links footer__links--secondary">
      <li class="footer__link footer__buttonwrap">
        <a class="footer__anchor button button--green" href="http://app.uphex.com/sessions/new">Pricing &amp; Signup</a>
      </li>
      <li class="footer__link">
        <a class="footer__anchor" href="http://app.uphex.com/sessions/new">Log In</a>
      </li>
    </ul>
    <ul class="footer__links footer__links--primary">
      <li class="footer__link">
        <a class="footer__anchor" href="#">About Us</a>
      </li>
      <li class="footer__link">
        <a class="footer__anchor" href="mailto:support@uphex.com">Contact Us</a>
      </li>
      <li class="footer__link">
        <a class="footer__anchor" href="/terms.html">Terms of Service</a>
      </li>
      <li class="footer__link">
        <a class="footer__anchor" href="/privacy.html">Privacy Policy</a>
      </li>
    </ul>
  </div>
</footer>
```
