# Header

A header is a block element that should be placed at the top of the page.

```
<header class="header">
  <div class="container">
    <h1 class="header__title">
      <a href="/">UpHex</a>
    </h1>
    <a class="header__link" href="http://app.uphex.com/sessions/new">Log In</a>
  </div>
</header>
```
